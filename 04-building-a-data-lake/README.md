# Command Line for Building a Data Lake

## Getting Started


Change Directory
```sh
cd 04-building-a-data-lake/workshop
```


For Running docker
```sh
docker-compose up
```


For Clear Terminal
```sh
clear
```

To shutdown, press Ctrl+C and run:

```sh
docker-compose down
```
